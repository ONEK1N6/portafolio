# Portafolio Profesional - Ingeniería de Sistemas

Este es un portafolio web moderno y responsivo creado con HTML5, CSS3 y JavaScript Vanilla. Diseñado para estudiantes que buscan prácticas preprofesionales.

## Estructura del Proyecto

```
portafolio/
├── index.html              # Archivo principal
├── assets/
│   ├── css/
│   │   └── styles.css      # Estilos y diseño responsivo
│   ├── js/
│   │   └── main.js         # Lógica de menú y scroll
│   ├── img/                # Carpeta para imágenes (perfil, proyectos)
│   └── documents/          # Carpeta para CV (PDF)
└── README.md
```

## Personalización

1.  **Información Personal:** Abre `index.html` y busca "Juan Pérez" para reemplazarlo con tu nombre. Actualiza los textos de "Sobre mí" y "Educación".
2.  **Imágenes:**
    *   Coloca tu foto de perfil en `assets/img/`.
    *   Actualiza las tarjetas de proyectos con capturas reales.
3.  **CV:** Coloca tu CV en formato PDF en `assets/documents/` y actualiza el enlace en la sección `home` del `index.html`.
4.  **Enlaces:** Configura los enlaces `href` de los iconos sociales (GitHub, LinkedIn) con tus URLs reales.

## Despliegue en GitHub Pages

1.  Sube este código a un repositorio en GitHub.
2.  Ve a la pestaña **Settings** del repositorio.
3.  Entra en la sección **Pages** (menú lateral izquierdo).
4.  En "Source", selecciona `Deploy from a branch`.
5.  En "Branch", selecciona `main` (o `master`) y la carpeta `/root`.
6.  Haz clic en **Save**. En unos minutos tu portafolio estará online.
